# Day 1 — Antigravity Setup and Cloud Run Deployment

## Codelabs Completed

1. Getting Started with Google Antigravity
2. Deploy from AI Studio to Cloud Run

## Goal

The goal of Day 1 was to get familiar with Google Antigravity and deploy a basic AI-generated web application to Cloud Run.

## What I Built

A formal frontend application with two buttons: snowflakes and balloons. When the snowflakes button was pressed, a handful of snowflakes fell from the top of the screen. When the balloons button was pressed, a handful of balloons floated up from the bottom of the screen. 

## Key Concepts

- Antigravity IDE basics
- Prompt-driven development
- Reviewing generated code
- Deploying an app to Cloud Run
- Understanding local vs cloud execution
